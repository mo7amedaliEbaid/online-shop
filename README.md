# Online Shop
By using firebase as a backend, I developed a flutter app for shoping online.

## State Management
✔️ Bloc Pattern <br />


## Features and components
✔️ Authentication with Email & Password         
✔️ Authentication with Google Account         
✔️ Cart           
✔️ Animations           
✔️ Categories screen             
✔️ Product details screen      
✔️ Intro Screen     
✔️ Bottom Nav Bar      
✔️ Products Screen      
✔️ Profile Screen

## Dependencies

**firebase**
- firebase_core:
- firebase_auth:
- cloud_firestore:
- google_sign_in:

**state management**
- bloc:
- equatable:
- flutter_bloc:

**common**

- flutter_staggered_grid_view:
